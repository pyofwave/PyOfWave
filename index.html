PyOfWave by alcinnz

Open Source Wave server and web client in Python (Used to be PyGoWave)

An implementation of Google-designed Wave communication platform (a merge of eMail, wiki, IM, and forums) built upon Twisted Python. It comes with a jQuery/JavaScriptMVC web client to add to your site.

Dependencies

Python 2.7+ Twisted
