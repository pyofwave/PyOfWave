PyOfWave is an implementation of Google-designed Wave communication platform (a merge of eMail, wiki, IM, and forums) built upon Twisted Python.

It comes with a jQuery/JavaScriptMVC web client to add to your site.

It is the active branch of the PyGoWave client, but is not functioning fully yet.

Dependencies

  • Python 2.7+
  • Twisted
